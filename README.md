🛡️ Платформа підтримки РХБ захисту
Централізований веб-портал для фахівців РХБ захисту, що об'єднує модулі моніторингу, розрахункові калькулятори та довідкову базу в єдиному інтерфейсі.

🚀 Основні модулі
Платформа побудована за модульним принципом "Hub-and-Spoke", де головна сторінка слугує точкою доступу до окремих інструментів:

Модуль 1: Стан РХБ обстановки — карти радіаційного та хімічного моніторингу (фактичні та прогнозні).

Модуль 2: База даних НХР — доступ до аварійних карток, рівнів впливу БОР та знаків небезпеки.

Модуль 3: Калькулятори РХБ захисту — оперативний розрахунок отриманих доз опромінення та часу перебування у зонах зараження.

Модуль 4: Довідкова інформація — метеодані, інструкції до приладів (ДП-5В, МКС-05 тощо) та ЗІЗ.

🛠 Технологічний стек
Мова: Python

Фреймворк: Streamlit (забезпечує адаптивність для мобільних пристроїв)

Дані: Pandas, Openpyxl

Візуалізація: Folium (в інтегрованих модулях)

📦 Розгортання та запуск
Запуск локально
Клонуйте репозиторій:

Bash
git clone https://github.com/vash-username/rkhbz-portal.git
Встановіть необхідні бібліотеки:

Bash
pip install -r requirements.txt
Запустіть додаток:

Bash
streamlit run app.py
Розгортання у Streamlit Cloud
Завантажте код на GitHub.

Підключіть репозиторій до Streamlit Cloud.

Платформа автоматично розгорне додаток та надасть публічне посилання.

📈 Масштабованість
Архітектура дозволяє додавати нові модулі через додавання блоків st.link_button або st.container у файл app.py. Система автоматично підлаштовує дизайн під екрани смартфонів.
